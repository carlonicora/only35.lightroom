Only35 Lightroom Plugin
=======================

Installation
------------
1. Open Lightroom Classic
2. Go to File > Plug-in Manager
3. Click the "Add" button at bottom left
4. Navigate to and select this folder (only35.lrplugin)
5. Click "Add Plug-in"

The plugin should now appear in your Plug-in Manager list.

Usage
-----
1. In the Library module, click "Publish Services" in the left panel
2. Click "Set Up..." next to "Only35"
3. Click "Login" to authenticate with your Only35 account
4. Create a Published Collection to start uploading photos

Troubleshooting
---------------
If you encounter issues:
- Check that you are logged in (Publish Services > Only35 > Edit Settings)
- Verify your internet connection
- Check Lightroom logs in ~/Documents/Adobe/Lightroom/Logs/

For support, visit: https://only35.app/help/lightroom
